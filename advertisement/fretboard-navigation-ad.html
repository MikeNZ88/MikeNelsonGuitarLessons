🎸 The Cross-String Formula

String Tuning Interval - Frets Moved Up = Frets to Move Back

No note names required! This calculation lets you find any fret position on adjacent strings using only numbers!

String tuning interval for each pair: E-A, A-D, D-G, B-E use 5 • G-B uses 4

E to A: A (5th fret) → C (8th fret) = 3 up, so 5-3=2 back = 3rd fret A
G to B: A (2nd fret) → C (5th fret) = 3 up, so 4-3=1 back = 1st fret B
This isn't a substitute for learning note names, but it can provide a way of thinking about options for where to find the next note. It uses a reference point and therefore helps to find the "next note" to play.
