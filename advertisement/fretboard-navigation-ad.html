üéØ The Core Concept
When you know a note on one string and want to find the same note on an adjacent string:
Frets moved on original string + Frets moved cross-string = 5
(Exception: G and B strings = 4)

üßÆ Two Simple Formulas

üìà Going to THINNER string
Cross = Move - 5
(or Move - 4 for G and B)
üìâ Going to THICKER string
Cross = Move + 5
(or Move + 4 for G and B)

"Move" = How many frets you moved on the original string
"Cross" = How many frets to move on the new string
